# Vision

## One-line pitch
A private parent-child co-op village life sim where value creation, saving, productive ownership, and civic contribution visibly shape the world.

## Emotional target
The game should create the feeling:
- I want something meaningful
- I cannot have it yet
- I need a plan
- I can make mistakes
- patience and good judgment unlock better life

## Pillars
1. Agency
2. Craft
3. Stewardship
4. Critical thinking
5. Society
6. Legacy

## Design principle
Each core mechanic should stay readable across the comprehension tiers defined in the constitution:
- Wonder
- Planning
- Strategy
- Systems
- Legacy

Example:
Apple tree
- wonder: pretty tree gives fruit every day
- planning: save now to buy it later
- strategy: recurring source of income changes better choices
- systems: tree output affects craft, trade, and household planning
- legacy: productive capital outlasts a single day of labor
