# Roadmap

## Milestone 0 - Canon and Repo Foundation
### Objective
Lock the repo into a stable documentation hierarchy, runtime boundary, and change-control baseline.

### Must-have Outcomes
- canon, design, and production docs are separated clearly
- stable ID and save compatibility rules exist
- Luanti runtime scaffold is split into data, systems, and runtime
- current content stays minimal and placeholder-safe

### Explicitly Banned Scope
- new gameplay loops
- extra professions or districts
- UI polish pushes
- public-online infrastructure

### Proof of Success
The repo can add new content without renaming core concepts or rethinking save structure.

## Milestone 1 - Orchard and Bridge
### Objective
Prove the smallest complete loop of production, saving, productive ownership, and civic contribution.

### Must-have Outcomes
- gather apples
- craft pies
- sell pies for coins
- save toward one apple tree
- contribute to one bridge project
- see visible bridge progress and completion
- receive simple NPC feedback

### Explicitly Banned Scope
- extra districts
- advanced AI NPCs
- tax systems
- inheritance mechanics
- public server support

### Proof of Success
A child can understand the loop quickly and feel that both the tree and the bridge matter.

## Milestone 2 - Shared Household Layer
### Objective
Make co-op feel meaningfully shared through household ownership and household decisions.

### Must-have Outcomes
- shared household storage or equivalent shared holdings
- clear distinction between personal and household ownership
- at least one household-level trade-off

### Explicitly Banned Scope
- full town politics
- profession explosion
- deep institution simulation

### Proof of Success
Two players can feel the difference between "mine," "ours," and "town."

## Milestone 3 - First District Expansion
### Objective
Prove additive growth by adding one new district without replacing the original game.

### Must-have Outcomes
- one new district with its own places and specialist
- one new project or civic structure
- one new content chain that reuses the same core loops

### Explicitly Banned Scope
- systemic rewrites of Orchard and Bridge
- old content obsolescence
- public-online progression layers

### Proof of Success
The new district feels like more game, not a different game.

## Milestone 4 - First True Judgment / Information Loop
### Objective
Introduce meaningful uncertainty and conflicting advice while preserving child readability.

### Must-have Outcomes
- hints, rumors, or records that influence choices
- at least one commission or dilemma with incomplete information
- visible consequence tied to trust or judgment

### Explicitly Banned Scope
- opaque simulation
- hard-to-read economic abstraction
- NPC complexity that outgrows the rest of the project

### Proof of Success
Players can make understandable mistakes for understandable reasons and learn from them.
