# V1 Vertical Slice

## Name
Orchard and Bridge

## Play loop
1. Gather apples
2. Craft pies
3. Sell pies
4. Earn coins
5. Choose:
   - spend immediately
   - save for tree
   - contribute to bridge
6. Buy apple tree
7. Tree yields apples over time
8. Bridge progresses visually as fund increases
9. NPCs comment on decisions

## Required systems
- inventory
- one crafting recipe
- one sell interaction
- coin tracking
- one productive asset
- one shared progress meter
- one map change after bridge completion
- one NPC feedback system

## Required content
Items:
- apple
- flour
- pie
- coin
- tree deed

World:
- apple source
- crafting station
- market stall
- bridge construction site

NPCs:
- baker
- builder

## Success test
A player should be able to feel:
- effort creates value
- saving beats impulse
- ownership changes tomorrow
- public contribution changes shared life
